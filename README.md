HeatMeet 🗓️🔥
HeatMeet es una aplicación multiplataforma (iOS, Android, Windows) diseñada para simplificar la organización de eventos grupales. Olvídate de los hilos de mensajes interminables para decidir cuándo quedar; visualiza la disponibilidad de todos tus amigos y encuentra el momento perfecto en segundos.

✨ Características principalas
Calendario Inteligente: Basado en Syncfusion Scheduler para una experiencia fluida.

Gestión de Disponibilidad: Marca tus huecos libres (días completos o horas específicas) con un solo toque.

Sincronización en Tiempo Real: Comunicación mediante Sockets para actualizaciones instantáneas entre usuarios.

Sistema de Grupos: Crea círculos de amigos y visualiza los eventos comunes de forma organizada.

Multiplataforma: Desarrollado con .NET MAUI para una experiencia nativa en cualquier dispositivo.

🛠️ Stack Tecnológico
Frontend: .NET MAUI (C# / XAML)

UI Components: Syncfusion MAUI Controls

Backend: Servidor en C# (Sockets / Multi-threading)

Base de Datos: PostgreSQL con Entity Framework Core

Data Format: JSON (System.Text.Json)
