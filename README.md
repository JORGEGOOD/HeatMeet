## 📖 Descripción

**HeatMeet** es una aplicación multiplataforma (iOS, Android y Windows) diseñada para simplificar la organización de eventos grupales.

Olvídate de los hilos interminables de mensajes para decidir cuándo quedar.  
Visualiza la disponibilidad de todos tus amigos y encuentra el momento perfecto en segundos.

---

## ✨ Características principales

### 📅 Calendario Inteligente
Basado en **Syncfusion Scheduler** para una experiencia fluida e intuitiva.

### ⏱️ Gestión de Disponibilidad
Marca tus huecos libres (días completos o franjas horarias) con un solo toque.

### 🔄 Sincronización en Tiempo Real
Comunicación mediante **Sockets** para actualizaciones instantáneas entre usuarios.

### 👥 Sistema de Grupos
Crea círculos de amigos y organiza eventos de forma clara y visual.

### 📱 Multiplataforma
Desarrollado con **.NET MAUI** para funcionar en cualquier dispositivo.

## 🛠️ Stack Tecnológico

```bash
Frontend        → .NET MAUI (C# / XAML)

Backend         → Servidor en C# (Sockets / Multi-threading)
Base de Datos   → PostgreSQL + Entity Framework Core
Data Format     → JSON (System.Text.Json)
